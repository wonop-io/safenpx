# Requirements Brief

- Map every M3 deliverable from `docs/milestones.md` to GitHub issues.
- Map every M3 acceptance criterion from `docs/milestones.md` to GitHub issues.
- Preserve and refine existing M3 tickets instead of duplicating them.
- Include scope, acceptance criteria, verification, dependencies, and references
  in the ticket set.
- Add an explicit M3 closeout/audit ticket.
