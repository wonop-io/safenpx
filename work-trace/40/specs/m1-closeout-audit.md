# M1 Closeout Audit

## Audit Checks

- Every M1 deliverable is complete or has a later-milestone follow-up.
- Every M1 acceptance criterion has direct code, test, fixture, or CI evidence.
- All M1 issues are closed before milestone closure.
- Latest `main` CI run is green.
- Local `just test` is green.

## Required Evidence

- Issue closeout table.
- Deliverable-to-evidence table.
- Acceptance-criteria-to-evidence table.
- Latest local verification command result.
- Latest GitHub Actions run URL.
- M1 open issue count before closure.

## Non-Goals

- Implement new resolver behavior.
- Expand M1 into M2 execution-closure work.
- Weaken the documented distinction between verified root artifact and verified
  execution closure.
