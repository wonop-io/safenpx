# Review Turn Port Spec

## Acceptance Criteria

- `playbooks/repository/review-turn.md` exists with valid playbook metadata.
- The playbook requires review against a prior commit or commit range.
- The playbook requires clear acceptance criteria before running red-team /
  blue-team / judge review.
- The playbook says re-reviews are limited to two.
- Repository playbook indexes link the new playbook.
- `just test` passes.
